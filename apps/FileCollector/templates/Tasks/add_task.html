Tasks

Here you can create a task or a task preset. Task presets can be used for recurring tasks and are a chill way to speed up your workflow.

Type what kind of work does your task involve most?

Start time if you set a start time the user will be reminded (in the future, not yet working) that he/she should start this task. If it's left empty the user can start whenever they want.

Deadline if you set a start time the user will be reminded (in the future, not yet working) that he/she a task should be finished at this deadline. The priority will increase as well. If it's left empty the user can finish whenever they want.

Description Let the user - or whoever reads this - know what this task was about. Something peticular the client wanted? Or maybe just an info to the other persons involved in this project.
